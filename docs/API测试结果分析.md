## ❌ 测试失败的问题（10个）

### 1. Intent接口参数问题（4个失败）

**端点**: `POST /api/v1/agents/intent`  
**状态码**: 422  
**错误**: `Field required in 'query'`  

**原因**: 测试脚本使用`json`传参，但接口期望`query`参数

**修复方案**:
```python
# 错误写法
json={"message": message}

# 正确写法
params={"message": message}
```
